# 🎵 Music Analytics Dashboard

A clean and interactive Google Sheets dashboard analyzing music data such as song popularity, genres, and listening patterns.  
This project demonstrates skills in data cleaning, KPI creation, pivot tables, and dashboard design.

---

## 📊 Overview

This dashboard provides insights into a small dataset of popular songs, including:

- Total plays  
- Most popular genres  
- Most played artists  
- Average song length  
- Visual charts showing listening trends  

The goal of the project is to practice data analysis and dashboard creation using Google Sheets.

---

## 📁 Dataset

The dataset contains the following columns:

- **Song** – title of the track  
- **Artist** – performer  
- **Genre** – music category  
- **Length (min)** – duration of the song  
- **Plays** – number of times the song was played
